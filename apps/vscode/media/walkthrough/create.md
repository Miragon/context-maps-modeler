# Create your first diagram

Context Maps diagrams live in `.cm` files (or `.cm.json`). This extension opens them in a
visual editor instead of showing raw text.

**Do this now:** click **Create diagram from example** in the step on the left. (You can also open
the Command Palette with `Cmd/Ctrl+Shift+P` and run **Context Maps: New Diagram from Example**.)

Then:

1. A save dialog appears — choose a folder and file name ending in `.cm` and confirm.
2. The file opens straight in the visual editor, pre-filled with a few bounded contexts and
   relationships you can rearrange and learn from.

Prefer a blank canvas? Run **Context Maps: New Empty Diagram** instead. Either way, any file
ending in `.cm` or `.cm.json` opens in this editor automatically from now on.
